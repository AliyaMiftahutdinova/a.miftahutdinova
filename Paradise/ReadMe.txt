На сравнение подаются 2 файла: Original.txt и Change.txt. Последовательность действий для получения из первогой файла второго записывается в файл Change.diff, находящийся в отдельной папке. На основе файла с разрешением diff с командной строки может быть запущена команда patch, которая тут же проведет изменения в старом файле. 

Пример:

Согласно моей иерархии папок, программа лежит в папке С:/Paradise/src, оба текстовых файла - C:/Paradise/files, diff-файл - C:/Paradise/change. Запуск в cmd команды patch будет выглядеть следующим образом:

С:\> patch < C:/Paradise/change/Change.diff; 

Выполнение последней команды перенесет разницу из нового файла в старый, обновив старый файл.

Примечание: Если Вы хотите увидеть последовательность действий для приведения старого файла к новому, сохраните ее в файл формата txt.
